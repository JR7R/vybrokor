# Vybrokor Release Notes v1

This is the first public release of Vybrokor documentation.

## Included in this release

- Public project statement
- Environmental purpose
- Open release direction
- Initial documentation

## Planned future additions

- Concept images
- Prototype photos
- Build notes
- CAD files after testing and finalization
- Additional file formats by request where practical

## Release intent

Vybrokor is being released openly to encourage real-world adoption of cleaner mechanical alternatives to chemical-heavy motorcycle tank cleaning methods.
